### Readme file to help you run the project

## Prerequesites

**Expo** - Install the latest version of Expo\
**Android Studio** - Install the latest version of Android Studio and set-up a virtual device if you wish to emulate the project on a virtual phone\
**Google Account** - ensure you already have a google account setup on the device you wish to run the app on\
**Npm** - Make sure you have setup node js and npm\
**Visual Studio Code** - if you wish to modify code or view multiple files at once , it is recomended to use VS Code as IDE\
**Expo GO** - Install expo go in your iOS or Android Device if you wish to emulate on physical device.


## Getting started

**Running the Code** \
-> Download the project from Code-Project folder \
-> Open the Project in VS Code \
-> open terminal and execute ``` npm i ``` to install node_modules folder with npm\
-> after successful installation , execute ``` npx expo start``` to start the expo server \
-> scan the QR Code on your device or press ```a``` to start android emulation in Android Studio\
-> *make sure Android Studio and virtual device are running before executing ```a``` command*\
-> use the App on your device or emulation\

***Note - Open App.js and install any dependencies (libraries) accordingly if any error occurs***
